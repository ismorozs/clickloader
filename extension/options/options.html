    Special Rules

    1. URL - URL of the page where thumbnails are located. It may not be a full URL, but it must start exactly as in the address bar and include at least the full domain name (e.g., http://www.example.com). It must be set if a full domain name has fewer or more than 3 terms 2. Thumbnail - CSS selector of the thumbnail (very rarely needs to be specified, the extension knows that usually it's something with "src" attribute) 3. Original Image - CSS selector of the image on the next page where the link leads to (for cases when it's a full-fledged HTML page) 4. Naming - how the image being downloaded will be named 5. Folder - folder name to save the image to (no need to fill out all inputs)
