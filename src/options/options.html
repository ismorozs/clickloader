    Special Rules

    1. URL - URL of the page you wish download the image from, may not be full, but must start exactly as in the address bar 2. Thumbnail - CSS selector of the thumbnail (very rare need to specify, the extension knows that usually it's something with "src" attribute) 3. Original Image - CSS selector of the image on the next page where the link leads to (for cases when it's a fullfleged html page) 4. Naming - how the image being downloaded will be named 5. Folder - folder name to save the image to (no need to fill out all inputs)
